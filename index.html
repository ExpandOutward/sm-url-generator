SM URL Generator

SM API URL Generator

This tool is only applicable to SAP Ariba Support Engineers. It generates a URL that can be used in to generate and push new payloads from SLP to a SAP system. Instructions for using this API can be found on this Wiki page. The Wiki can only be accessed by internal SAP Ariba employees authenticated into the SAP internal network.

SM Advanced View URL Generator

This tool is use to generate a URL to access Advanced View specifically for SLP Projects. You must be logged into the customer site respective to the URL before pasting the URL into your browser. Otherwise, you'll be taken to the login screen instead of the project.
